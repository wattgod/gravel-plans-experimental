# Unbound Gravel 200 Landing Page - Elementor Export

## Overview

This directory contains the complete Elementor page export for the **Unbound Gravel 200 race landing page**. This is the top-of-funnel marketing content that drives traffic to the TrainingPeaks training plans.

## File

- **`elementor-unbound-200.json`** - Complete Elementor page export (JSON format)

## What This Page Contains

The landing page includes the following sections:

1. **Hero Section**
   - Race name and tagline
   - Primary call-to-action

2. **Course Vitals**
   - Distance, elevation, terrain details
   - Key race statistics

3. **Ratings Breakdown**
   - Difficulty ratings
   - Technical challenge ratings
   - Endurance requirements

4. **Training Plans Cards**
   - All 15 training plans displayed as cards
   - Each card links directly to TrainingPeaks marketplace
   - Organized by tier (Ayahuasca, Finisher, Compete, Podium)

5. **Logistics Section**
   - Race date, location, registration info
   - Additional race details

## How to Import to WordPress/Elementor

### Prerequisites
- WordPress site with Elementor Pro installed
- Elementor version 3.0+ recommended

### Import Steps

1. **Log into WordPress Admin**
   - Navigate to your WordPress dashboard

2. **Go to Templates → Import Templates**
   - In Elementor, go to **Templates** → **Import Templates**

3. **Upload the JSON File**
   - Click **Choose File** or drag and drop
   - Select `elementor-unbound-200.json`
   - Click **Import Now**

4. **Assign to Page**
   - After import, the template will appear in your Elementor templates
   - Create a new page or edit existing page
   - Click **Edit with Elementor**
   - Click the folder icon (Template Library)
   - Find "Unbound Gravel 200" template
   - Click **Insert**

5. **Update Permalink**
   - Set the page slug to match your desired URL (e.g., `/unbound-gravel-200`)
