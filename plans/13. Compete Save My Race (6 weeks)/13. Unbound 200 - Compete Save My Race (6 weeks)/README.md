## Quick Start

1. **Review Week 1 assessment results** to identify your limiter
2. **Choose your block path:**
   - VO2max Block: Use files ending in `_VO2max`
   - Threshold Block: Use files ending in `_Threshold`
   - Durability Block: Use files ending in `_Durability`
3. **Delete the other two block options** from the generated files folder
4. **Upload to TrainingPeaks** (test with one file first)

## Key Modifications for Unbound 200

1. **Heat Training** (Weeks 2-5, all quality sessions) - 3-tier system
2. **Aggressive Fueling** (60-90g carbs/hour on all long rides)
3. **Dress Rehearsal** (9-hour ride Week 3 Saturday)
4. **Robust Taper** (Week 6 volume reduced)
5. **Gravel Grit** (Week 6 race day mental training)
6. **Cadence Work** (high/low cadence on all intervals)
7. **Rhythm/Loaded Intervals** (Week 5 race simulation)
8. **G-Spot Terminology** (87-92% FTP replaces Sweet Spot)

See `COMPETE_SAVEMYRACE_UNBOUND200_MODIFICATIONS.md` for complete details.
