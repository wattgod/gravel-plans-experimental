# Unbound Gravel 200 Marketplace Descriptions

## Structure

This directory contains **60 tier-specific marketplace descriptions** for Unbound Gravel 200 training plans, organized by tier:

- `ayahuasca/` - 15 variations (0-5 hrs/week)
- `finisher/` - 15 variations (8-12 hrs/week)
- `compete/` - 15 variations (12-18 hrs/week)
